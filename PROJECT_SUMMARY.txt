🚀 DEPLOY TO VERCEL (5 MINUTES):

1. Push to GitHub:
   git init
   git add .
   git commit -m "Initial commit"
   git push origin main

2. Go to vercel.com → "New Project"

3. Import your GitHub repo

4. Add environment variables:
   NEXTAUTH_SECRET=<generate-random>
   NEXTAUTH_URL=https://your-app.vercel.app

5. Create Vercel Postgres database:
   Storage → Create → Postgres (Free tier)

6. Deploy!

Your app is live at: https://your-app.vercel.app

Detailed guide: See DEPLOY_TO_VERCEL.md

🎓 EDUCATIONAL CONTENT:

11 Architectural Elements:
  1. Altar          - Sacred structures in religious buildings
  2. Apse           - Semi-circular terminations
  3. Bell Tower     - Towers designed to house bells
  4. Column         - Classical orders (Doric, Ionic, Corinthian)
  5. Dome (Inner)   - Interior dome structures
  6. Dome (Outer)   - Exterior dome structures
  7. Flying Buttress - External support structures
  8. Gargoyle       - Decorative water spouts
  9. Portal         - Ornamental doorways
  10. Stained Glass - Colored glass artworks
  11. Vault         - Arched ceiling structures

Learning Modules:
  • Introduction to Heritage Architecture
  • Detailed Element Studies
  • Architectural Styles & Periods
  • Cultural Preservation Techniques

Assessment Types:
  • Multiple choice questions
  • True/False questions
  • Image identification
  • Matching exercises
